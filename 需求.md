基于C++/QT实现一个贪吃蛇的桌面小游戏。
功能需求：
蛇自动移动
上下左右控制方向
吃到食物增长
撞墙/撞自己游戏结束
技术约束：
使用 C++
使用 Qt（Widgets，不使用 QML）
使用 CMake
不使用 .ui 文件
交互要求：
键盘控制（↑ ↓ ← →）
实时刷新（定时器驱动）
最终效果：
游戏窗口大小（如 800x500），界面左侧是游戏区域，右侧是一个侧边栏提示用户
游戏规则，已经计分板，还有重新开始，暂停等按钮
合适的背景色 + 绿色蛇 + 红色食物
显示分数
游戏结束弹窗
工程结构：
在SnakeGame这个文件夹下进行代码生成，代码放到SnakeGame/src文件夹内，然
后生成的exe、pdb等结果产物，均放到SnakeGame/bin文件夹内，并在SnakeRewind
文件夹下生成一个readme.md文档，readme文档要包含简单的项目介绍、技术栈、架构
设计思路、源码如何运行等等
代码的生成，参考SnakeGame/doc/coding_rules.md文件的编程规范。
