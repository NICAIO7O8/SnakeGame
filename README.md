# 贪吃蛇桌面小游戏 (Snake Game)

## 项目介绍
这是一个基于 C++17 和 Qt Widgets 开发的经典贪吃蛇桌面游戏。该工程严格遵循代码规范，实现了逻辑与 UI 分离，提供了纯手写构建界面的完整流程。

## 技术栈
- **语言**: C++17
- **框架**: Qt (Widgets)
- **构建工具**: CMake

## 架构设计思路
应用主体包裹在 `snakegame` 命名空间中。依据设计规范，禁用了 `.ui` 文件，纯使用代码进行 UI 组件布局。
1. **GameEngine (游戏引擎核心)**: 负责维护蛇的身段（坐标）、食物的生成、得分计算及方向约束判断，使用 Qt 定时器驱动游戏进度，完全剥离视图依赖。
2. **GameWidget (游戏渲染视图)**: 游戏界面画布。负责捕获用户的键盘事件（上下左右），重写 `paintEvent` 绘制当前的网格背景、绿色的蛇和红色的食物。
3. **MainWindow (主界面控制器)**: 组合视图与右侧交互侧边栏，提供暂停、重启与规则说明。它接管 GameEngine 抛出的信号（例如加分、Game Over），并且通过弹窗反馈游戏状态。

## 源码运行方式
1. 打开 VS Code（或你喜欢的终端）。
2. 使用 CMake 配置项目（例如使用 CMake Tools 扩展，或命令行：`cmake -S . -B build`）。
3. 编译工程：`cmake --build build`。
4. 生成的结果（`SnakeGame.exe` 及相关产物）将被输出到工程根目录下的 `bin` 文件夹。
5. 运行 `bin/SnakeGame.exe` 开始游戏。
